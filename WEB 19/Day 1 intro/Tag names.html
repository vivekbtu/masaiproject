b tag

The b HTML element is used to draw the reader's attention to the element's content's, whuch are not otherwise granted special importance. This was formerly known as the Boldface element, and the most browsers still draw the text in boldface

i tag

The i HTML element is used to display normal text in from of italics.

u tag

The u HTML element is used to span of inline text which should be used in a way that indicates that it has a non-textual annotation. This is used by default as a simple solid underline.

select tag

The select HTML element is used to control that a menu options.

button tag

The button HTML element is used to an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology. Once activated, it then performs a programmable action, such as submitting a form or opening a dialog.
